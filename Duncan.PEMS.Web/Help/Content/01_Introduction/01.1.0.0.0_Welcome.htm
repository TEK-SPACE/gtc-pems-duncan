CivicSmart's Parking Enterprise Manager System (PEMS) is a comprehensive meter / sensor / inventory management, reporting, and data analysis application that allows you to efficiently manage all aspects of your on and off street parking meter program. PEMS is a powerful tool that provides:

  • High level management reports
  • Detailed transaction and event based auditing for each payment type (cash, credit card, smart card, pay-by-cell)
  • Real-time meter and space occupancy status monitoring
  • Exception reporting
  • Data extraction for external reporting or interfacing.

Each CivicSmart parking meter and sensor gateway is equipped with a wireless communication module and communicates with PEMS using a secure and proven mobile / wireless technology. This communication strategy facilitates real-time connectivity between each device and a central server to allow transmission of alarms / data, and provides a complete picture of the parking operation.
