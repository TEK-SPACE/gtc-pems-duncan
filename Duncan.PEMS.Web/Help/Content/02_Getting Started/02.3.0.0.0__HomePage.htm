Home Page 

The PEMS Home page provides a simple, easy to navigate interface for access to each of the available reporting and analysis modules. The welcome screen displays immediately after you log in.

 

News and Announcements

The home page also includes a news and announcements section which will display the latest updates and changes made in PEMS for the active client. This is displayed within the main body of the PEMS home page.

Management Pane

The main feature of the home page is the Management Pane on the right side of the screen.This section consolidates all of the PEMS management modules into a single set of tabs. Within each module you can perform a data base inquiry.

The modules visible in the management pane depend on the level of access granted to the user through the assigned role the user profile. The user will only be able to see modules that the PEMS administrator has granted access to in the user's profile.

Toolbar

The homepage includes a simple toolbar, located in the top right corner, for non-management related actions. This also displays the current time and date in your area, as well as the user name of the user who is currently logged in. The other functions are as follows:

  • Clicking on the button to returns the user to the home page.
  • Clicking on the button opens the online help file.
  • Clicking on LOGOUT AS logs the user out of PEMS.
