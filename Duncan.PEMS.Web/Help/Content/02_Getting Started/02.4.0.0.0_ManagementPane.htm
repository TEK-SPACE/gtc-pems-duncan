Management Pane

All PEMS features are organised into several modules which you access via the Management Pane located on the left side of the PEMS web page.

 

PEMS Management Modules

PEMS modules are organised by function.

Module Description

Asset Management

Search and review details of assets deployed in the field (meters, sensors, cash boxes, etc.).

Event Management

Search and review details of events (alarms, transactions, etc.) recorded in PEMS.

Alarm Management

Search and review details of any alarms raised by assets deployed in the field.

Transaction Management

Search and review transaction details reported by assets deployed in the field.

Collection Management

Search and review all revenue activity reported by assets deployed in the field.

Discounts Management

Manage and review details relating to all discount scheme accounts and payments.

Reports

Run various pre-defined and ad hoc reports.

Geographic Reports

Search and review inventory, space by demand type and operational statuses of assets in a geographical map by site location.

User Profile

Update and edit your personal user profile.

Administration

Used by the local PEMS administrator to manage user profiles, add new users, and create new roles.

Selecting a Module Function

  1. Click on a module tab name, or the symbol to the right of the module name, to expand a list of functions for that module.
  2. Click on the link for the target function to display an inquiry screen in the main section of the web page.
  3. Clicking on the module name again, or clicking the symbol. will collapse the list of module functions.
  4. Click collapse at the top of the management pane to make more room to view PEMS data on the screen . This will hide the management pane labels but the management pane icons will still be visible so you can navigate to other modules.

You can always tell which module you are currently working in because the active tab is highlighted with a lighter colour in the management pane. A breadcrumb trail at the top of the page will also indicate which screen is currently active.The Management Pane will always be visible on the left side of the web page to allow you to easily navigate from one module to another.

The filter options are expanded by default. To collapse the filter fields so you can see more of the inquiry results on the screen click on the button.
