PEMS Inquiries

The primary activity you perform in PEMS is a simple database inquiry. When you initially select any module option in the Management Pane, the page will display an array of filters which allow you to drill down to the specific data you are interested in viewing. The filter fields displayed vary from module to module, but the procedure for applying the filters is identical across all modules.

Once you apply the appropriate filters, PEMS will display the inquiry results in a table below the inquiry filter fields. For select modules, the initial inquiry results display in a summary view, and you have the option of selecting other views that highlight different aspects of the data.

 

To Apply Filters to an Inquiry

  1. Enter the parameter(s) you want to use as a filter(s) by either selecting from the field drop down lists or manually entering the object name/number in the filter fields (refer to the specific module sections in the Help file for field definitions).
  2. Click on the button to see the Inquiry Results table.  

To Change the Inquiry Results View:

  1. Select the target view from the Views: drop down list. The screen will automatically refresh and display the new view.
  2. Use the page navigation bar at the bottom of the window to scroll through the inquiry.
The filter options are expanded by default. To collapse the filter fields so you can see more of the inquiry results on the screen click on the button.
