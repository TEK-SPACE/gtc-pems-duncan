Asset Inquiry Filters

After selecting the Asset Management module in the Management Pane you may filter your asset inquiry using the following database fields:

Field Name Description
Asset Type A specific type of hardware located in the field.
Asset ID A unique identifier for each discrete asset in PEMS.
Operational Status Hardware will either be operational or non-operational based on the types of alarms that the system is currently raising or other special conditions (events, maintenance, etc.).
Asset Name A unique name for each discrete asset in the system.
Location Filter Select the GPS or Location Type option. Allows you to filter by specific GPS coordinates (longitude/latitude) or by a location type (area, zone, etc.).

 

Asset Inquiry Views

The Asset Management module provides four different views of the asset inquiry data.

Summary View

Provides all of the basic asset location and status details.

Configuration View

Provides all of the asset's software configuration details.

Occupancy View

Provides the operational status and occupancy information for space assets.

Functional Status View

Provides the alarm status and target repair time for the asset.
