Add Asset

New assets (gateways, multi space meters, and sensors) are added to PEMS from the Asset Inquiry screen.

 

To Add a New Asset

  1. Click on the button in the Asset Inquiry screen.
  2. Select the radio button next to the new asset

Additional information for specific asset types (different options are displayed depending on which radio button is selected) ...

  • Multi space meters - Enter the starting and ending bay (space) numbers for the meter
  • Sensors - Select the associated parking space and gateway
  • Single space meters - Enter the relevant details within the asset information and configuration detail sections.
  1. Click on the button.
  2. Enter the relevant details within the asset information and configuration detail sections.
  3. Click on the button.

To Import New Assets

  1. Click on the button in the Asset Inquiry screen.
  2. Create an import file using the instructions provided in the Update Assets window.
  3. Select the appropriate asset type from the Asset Type drop down list.
  4. Click on the button and browse to where the import file is saved.
  5. Click on the button to complete the import.
