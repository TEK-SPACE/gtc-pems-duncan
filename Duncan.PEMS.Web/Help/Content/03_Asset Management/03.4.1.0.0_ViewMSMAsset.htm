View Multi Space Meter Asset

The Multi Space Meter Asset view allows you to view and modify asset information fields. In addition, you may also add time to a multi space meter.

 

To View Multi Space Meter Asset Details

  1. Perform an asset inquiry for a multi space meter.
  2. Click on the meter's Asset ID in the Asset Inquiry Results table.
  3. Click on View Spaces to navigate to the space assets associated with the meter asset.
  4. Click on the button to return to the inquiry results.

To Add Time to a Multi Space Meter

  1. Click on the button in the View Asset screen (multi space meter).
  2. Select the target bay (space) that you want to add time to.
  3. Enter the amount of time to add.
  4. Click on the button.
  5. PEMS will display a message confirming that time has been added to the meter.
  6. Click on the button to return to the View Asset screen.

To Update a Multi Space Meter Asset

  1. Perform an asset inquiry for a multi space meter.
  2. Select the asset you want to update by clicking on its Asset ID in the Asset Inquiry Results table.
  3. Click on the button.
  4. Make the necessary edits to the information on the Asset page.
  5. Click on the button to make changes to spaces associated with the meter, if necessary.
  6. Select the appropriate inventory status from the Inventory Status: drop-down list.
  7. Click the button to return to the View Asset screen.
