Manage Assets

The Manage Assets module allows you to monitor and investigate activity related to any asset deployed in the field. This activity includes alarms raised by meters, periodically scheduled communications, and other meter events.

  • The Alarm Management function displays all alarms raised by assets in the field that are still active and require the attention of a technician. To drill down to the events you want to focus on you can apply several Alarm Filters in a variety of combinations. After applying the filters PEMS will display the Alarm Inquiry results.
  • The Event Management function provides the ability to drill down to the details of specific events (alarms, transactions, communications, etc.) relating to any asset in the system. To drill down to the events you want to focus on you can apply several Event Filters in a variety of combinations. After applying the filters PEMS will display the Event Inquiry results.
