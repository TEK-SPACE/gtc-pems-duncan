Event Inquiry Filters

After selecting the Event Management module in the Management Pane you may filter your event inquiry using the following database fields.

Field Name Description
Date Range Limits the inquiry results to events falling within the date range selected.
Time Type Narrows the inquiry to a specific time of the week (day, weekend, etc.).
Asset Type A specific type of hardware located in the field.
Asset Identifier Select Asset Name or Asset ID (the data field directly to the right will toggle between the two options.
Asset ID A unique identifier for each discrete asset in the system.
Event Identifier Select Event Class (a specific group of events) or Event Code (a specific type of event) and the data field directly to the right will toggle between the two options.
Location Type Select a location type (a geographical grouping of hardware in the field) and the data field directly to the right will display a field for that type.

Functional Status Type

Only available in the Functional Status view

Provides details on several operational characteristics of certain assets. Select one of the available functional status types from the drop down list and the value input field directly to the right will change to an appropriate parameter filter.

 

Event Inquiry Views

The Event Management module provides seven different views of the event inquiry data.

Summary View

The summary view is displayed by default when you perform an event inquiry. This view lists all events falling within the inquiry parameters and identifies the individual assets and their locations.

Functional Status

The functional status view will list all functional status types, e.g. software version, falling within the inquiry parameters. This view also adds two extra filter fields to the inquiry screen.

Alarms

The alarms view lists all alarms falling within the inquiry parameters and identifies such information as the source, severity and notification time.

Connections

The connections view is used to check the meter connection history for trouble shooting purposes.

Transactions

The transactions view is used to check payment amounts and credit card status.

Collection Comm

The collection comm view displays the amount collected from each cash box as well as the old/new cash box serial numbers.

Collection CBR

The collection CBR view displays the removal and insertion times for cash box collections and the difference between the amount read by the reader station and the amount counted and entered by the cash collection crew.
