Event Inquiry Views

After applying filters and refreshing the event inquiry, the inquiry results will display in the table at the bottom of the screen. By default, PEMS will display the Summary view.

To change event inquiry view:

  1. Select the target view from the Views: drop down list. The screen will automatically refresh and display the new view.
  2. Use the page navigation bar at the bottom of the window to scroll through the inquiry.
