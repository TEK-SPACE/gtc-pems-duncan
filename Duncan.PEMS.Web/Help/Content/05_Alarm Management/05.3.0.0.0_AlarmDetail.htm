Alarm Detail

From the alarm inquiry results table you can select an individual alarm and view its details in the Alarm Detail screen.

 

To view Alarm Details

  1. Perform an alarm transaction inquiry.
  2. Click on an alarm ID in the Alarm Inquiry Results table.
  3. Click on the button to return to the inquiry results.

To Clear an Alarm

  1. Click on the button in the alarm details screen.
  2. Select a Resolution code.
  3. Enter the date and time the alarm was cleared.
  4. Select the Technician ID of the tech that cleared the alarm.
  5. Enter any relevant notes regarding the alarm (if applicable).
  6. Click on the button.
