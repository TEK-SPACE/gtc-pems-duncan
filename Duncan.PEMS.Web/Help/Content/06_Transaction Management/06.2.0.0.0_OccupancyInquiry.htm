Occupancy Inquiry Filters

After selecting Occupancy Inquiry (Transaction Management module) in the Management Pane you may filter your inquiry using the following database fields

Field Name Description
Arrival Date Range Date range to include in the inquiry.
Asset Type A specific type of hardware located in the field.
Asset ID A unique identifier for each discrete asset in the system.
Asset Name A unique name for each discrete asset in the system.
Occupancy Status Indicates if space is occupied or non-occupied.
Operational Status Indicates if the hardware is operational or non-operational.
Non-Compliant Status Specifies if non-compliance is due to overstaying in a space following paid time expiration or due to failure to initially purchase time.
Time Type Narrows the inquiry to a specific time of the week (day, weekend, etc.).
Location Type Select a location type (a geographical grouping of hardware in the field) and the data field directly to the right will display a field for that type.

 

Viewing the Occupancy Inquiry

After applying filters and refreshing the occupancy inquiry, the results will display in the table at the bottom of the screen.
