Collection Route Configuration Detail

The Collection Route Configuration detail view displays how the collection route is organised and set up to operate. The detail screen also provides a list of all meters assigned to the collection route.

 

To View Configuration Details

  1. Perform a collection route inquiry.
  2. Click on the collection route name in the inquiry results table.
  3. Click on the button to return to the inquiry results.

To Copy a Collection Route Configuration

  1. Click on the button.
  2. Enter a name for the new collection route.
  3. Modify the collection route configuration details as necessary.
  4. Click on the button.
