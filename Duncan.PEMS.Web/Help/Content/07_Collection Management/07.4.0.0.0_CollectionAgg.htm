Collection Aggregation Inquiry Filters

After selecting Collection Aggregation (Collection Management module) in the Management Pane you may filter your inquiry using the following database fields:

Field Name Description
Collection Date Range Limits the inquiry results to collections within the date range selected.
Route ID A unique identifier for a specific group of meters.
Route Name A unique name for a specific group of meters.
Meter ID A unique identifier for each meter.
Meter Name A unique name for each meter.
Location Type A geographic grouping of hardware in the field.
Variance A difference between the total collected by the vendor and the total collected from the meter.

 

Viewing the Collection Aggregation Inquiry

After applying filters and refreshing the collection aggregation inquiry, the results will display in the table at the bottom of the screen.
