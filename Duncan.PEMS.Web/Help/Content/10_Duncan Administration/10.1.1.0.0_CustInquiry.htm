Customer Inquiry

A customer inquiry allows you to select a specific PEMS customer and review the details of their account. From the inquiry window you can update the customer record or add a new customer.

To perform a customer inquiry:

  1. Enter the customer name or customer ID in the inquiry screen.
  2. Click on the button to see the inquiry results.
     

 

The filter options are expanded by default. To collapse the filter fields so you can see more of the inquiry results on the screen click on the button.
The customer name or ID must match how the customer was set up in PEMS (not case sensitive) in order to get inquiry results. If you leave the Customer Name and Customer ID fields blank the inquiry will return all customers in PEMS.
