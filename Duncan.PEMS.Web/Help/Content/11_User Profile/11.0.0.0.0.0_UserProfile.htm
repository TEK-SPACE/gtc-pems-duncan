User Profile

Each PEMS user has their own unique profile which determines their system access. When logged into PEMS Users may change their passwords and update their personal information and security questions.

 

To Edit Your User Profile

  1. Click on User Profile in the management pane.
  2. Edit any fields that need to be corrected or updated.
  3. Click on the button.

To Change a User Password

  1. Click on User Profile in the management pane.
  2. Enter the new password in the New Password field and then again in the Confirm Password field.
  3. Click on the button.
