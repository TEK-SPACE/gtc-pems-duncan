User Management

A user profile controls what modules and functions a user can access and what types of changes the user can make on the system. A user profile should be created for each person who will need to have access to PEMS. Only an administrator can create or modify a user profile.

 

To Apply Filters to a User Inquiry

After selecting User Management (Administration module) in the Management Pane you may filter your user inquiry using the following database fields:

Field Name Description
Username Username assigned by PEMS used to log into the system.
First Name The user's first name.
Last Name The user's last name.
Status A user's PEMS status is set to active, suspended, or terminated.
Role Name A object assigned to a user which provides the user with a defined set of rights in PEMS.

 

To Create a User Profile

  1. Click on the button on the User Inquiry screen.
  2. Enter the user identification information for the user.
  3. Select a role for the user.
  4. Click on the button.
PEMS will automatically assign a user name in the following format: [first initial][middle initial][last name].
PEMS will automatically assign new users a password of Duncan1!. The user will be required to change their password at the first login.
The user must select two security questions when they first log on.

To Edit a User Profile

  1. Click on the target user name in the user inquiry results table.
  2. Click on the button in the User Details screen.
  3. Edit any fields that need to be corrected or updated.
  4. Click on the button.

To Change a User Password

  1. Click on the target user name in the user inquiry results table.
  2. Click on the button in the User Details screen.
  3. Enter the new password in the New Password field and then again in the Confirm Password field.
  4. Click on the button.

To Modify a User Profile

  1. Select the user(s) to be modified by checking the box next to the name(s) in the user inquiry results table.
  2. Select the modification you would like to make from the Action: drop-down list.
  3. Select the status or role change in the Bulk User Update screen.
  4. Click on the button.
