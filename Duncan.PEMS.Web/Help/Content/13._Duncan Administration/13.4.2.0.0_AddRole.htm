Add a Role

New roles for each customer can be created as needed from the Role Inquiry screen.

To Add a New Role:

  1. Click on the button.
  2. Enter a name for the new role in the Role Name field.
  3. Check mark the box next to each right the new role will have (see right definitions below).
  4. Click on the button to create the new role.
